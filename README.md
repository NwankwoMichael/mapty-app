Mapty is a workout tracking app built with JavaScript and Leaflet.js.  
It allows you to log running and cycling workouts on an interactive map.

## Features

- Log workouts with distance, duration, cadence/elevation
- Calculate pace (running) and speed (cycling)
- Render workouts on a map using Leaflet
- Persist data with localStorage
- Edit and delete workouts
